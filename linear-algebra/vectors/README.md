# Vectors: A Tiny Book

This is a rereadable guide for rebuilding vector intuition without starting from memorized formulas.

The central idea:

> A vector space is a set of reachable places created by scaling and summing direction vectors.

Read in order:

1. [Points, Vectors, And Anchors](01-points-vectors-anchors.md)
2. [Linear Combination, Span, And Rank](02-span-rank-dimension-mobility.md)
3. [Basis And Matrices](03-basis-and-matrices.md)
4. [Symmetry Before Dot Product](04-symmetry-before-dot-product.md)
5. [Dot Product And Orthogonality](05-dot-product-orthogonality.md)
6. [Projection](06-projection.md)
7. [Local Coordinates For Shapes](07-local-coordinates-for-shapes.md)
8. [Formula Sheet](formula-sheet.md)

Language note:

- **global coordinates** means coordinates written in the usual world basis, like $i,j,k$.
- **local coordinates** means coordinates written in a chosen shape/basis, like $u,v,n$.
- **rank** means dimension mobility: how many independent directions are really available.
