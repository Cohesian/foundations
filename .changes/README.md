# Changes (local)

This folder is for **your machine only**. Everything here is gitignored except this README.

Use it for:

- Rollout / migration plans while a refactor is in flight
- PR title and description drafts (copy-paste into GitHub)
- Scratch checklists you do not want in the k-graph or `docs/`

**Permanent spec** stays in the repo: [docs/KGRAPH.md](../docs/KGRAPH.md), structure under [k-graph/](../k-graph/), content under [data/documents/](../data/documents/).

## Suggested filenames

| File | Purpose |
|------|---------|
| `pr-draft.md` | PR body ready to paste |
| `rollout-*.md` | Step-by-step migration notes |
| `todo-*.md` | Personal follow-ups |

Delete or archive files when the work is merged.
