# Contributing

This repository is personal and exploratory. Suggestions and discussions are welcome when they align with the project’s purpose.

---

## What helps

- Improving explanations
- Adding diagrams and examples
- Finding related structures
- Suggesting alternative perspectives
- Adding computational experiments
- Identifying invariants
- Connecting ideas across domains

The best contributions do not only add information — they **reveal structure**.

See [PRINCIPLES.md](PRINCIPLES.md) for how material is usually written, and [KGRAPH.md](KGRAPH.md) for where new files should live.

---

## Development process

Work is developed with AI-assisted tools (e.g. Codex). Ideas, mathematical direction, conceptual framing, and final judgment come from Cohesian; AI helps polish explanations, structure papers, write code, generate notebooks, and explore examples.

AI is a tool for expression, experimentation, and refinement — not a replacement for understanding.
